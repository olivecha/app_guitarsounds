Les courbes de l'enveloppe logarithmique des différentes bandes de fréquences sont tracées de façon superposée. 

Cette analyse permet de voir comment l'amplitude à certaines fréquences varie dans l'attaque et l'amortissement du son. 

Puisque l'amplitude des bandes à haute fréquence est beaucoup moins élevée que celle des basses fréquences, une échelle logarithmique est utilisée pour l'amplitude.

La fonction "Écouter les bandes de fréquence" peut être utilisée pour entendre les sons correspondant aux différentes courbes. 
Voir l'aide associée pour en apprendre d'avantage sur les différentes bandes de fréquence.
